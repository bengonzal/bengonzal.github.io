ASCII Project

Status: Haitus

The purpose of this project is to create ascii art from a given image. Further explanation and examples can be seen down below, but the gist of this project is to take an image, apply a grayscale filter, map pixel values from the grayscale picture to an ascii charset, and recreate the image entirely out of ascii characters.
